Fuzzy C-means

Pengertian Fuzzy C-Means (FCM)

Fuzzy C-Means (FCM) adalah algoritma clustering berbasis fuzzy logic, di mana satu data dapat menjadi anggota dari lebih dari satu cluster dengan derajat keanggotaan (membership value) tertentu, yaitu nilai antara 0 dan 1.

Berbeda dengan K-Means yang bersifat hard clustering (satu data = satu cluster), FCM menerapkan prinsip soft clustering, karena kenyataannya, banyak data yang berada di batas antar kelompok dan tidak bisa diklasifikasikan secara mutlak.

Inti Konsep FCM

  • Kita ingin mengelompokkan n data ke dalam c buah cluster.

  • Setiap data memiliki derajat keanggotaan (membership degree) terhadap masing-masing cluster.

  • Derajat keanggotaan tersebut jumlahnya harus 1 untuk tiap data (misalnya: 0.2 ke cluster A, 0.8 ke cluster B).

  • Derajat keanggotaan ditentukan berdasarkan jarak data ke pusat cluster: semakin dekat, semakin tinggi nilainya.

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

# Data input
data = np.array([
    [1, 2],
    [2, 3],
    [3, 4],
    [6, 8],
    [7, 8]
])

# Parameter FCM
m = 2
c = 2
epsilon = 1e-5
max_iter = 100

# Inisialisasi derajat keanggotaan secara acak, tapi distandarkan (jumlah per baris = 1)
np.random.seed(42)
u = np.random.dirichlet(np.ones(c), size=len(data))

# Fungsi menghitung centroid
def calculate_centroids(u, data, m):
    num_clusters = u.shape[1]
    centroids = np.zeros((num_clusters, data.shape[1]))
    for j in range(num_clusters):
        um = u[:, j] ** m
        numerator = np.dot(um, data)
        denominator = np.sum(um)
        centroids[j] = numerator / denominator
    return centroids

# Fungsi update derajat keanggotaan
def update_membership(data, centroids, m):
    n = data.shape[0]
    c = centroids.shape[0]
    u_new = np.zeros((n, c))
    for i in range(n):
        for j in range(c):
            denom = sum(
                (np.linalg.norm(data[i] - centroids[j]) / np.linalg.norm(data[i] - centroids[k])) ** (2 / (m - 1))
                for k in range(c)
            )
            u_new[i, j] = 1 / denom
    return u_new

# Proses iteratif FCM
for iteration in range(max_iter):
    centroids = calculate_centroids(u, data, m)
    u_new = update_membership(data, centroids, m)
    if np.linalg.norm(u - u_new) < epsilon:
        break
    u = u_new

# Hasil akhir
final_membership = u
final_labels = np.argmax(final_membership, axis=1)

# Tampilkan tabel keanggotaan
df_result = pd.DataFrame(data, columns=["x1", "x2"])
df_result["u_c1"] = final_membership[:, 0]
df_result["u_c2"] = final_membership[:, 1]
df_result["Cluster"] = final_labels + 1  # cluster dimulai dari 1
print(df_result)

# Visualisasi hasil clustering
colors = ['red', 'blue']
for i in range(c):
    plt.scatter(data[final_labels == i][:, 0], data[final_labels == i][:, 1], 
                color=colors[i], label=f'Cluster {i+1}')
plt.scatter(centroids[:, 0], centroids[:, 1], marker='X', s=200, color='black', label='Centroid')
plt.title("Hasil Clustering Fuzzy C-Means")
plt.xlabel("x1")
plt.ylabel("x2")
plt.legend()
plt.grid(True)
plt.show()
